computeGlobalBBox()

Compute global bounding box and translate and scale every object in proportion of global bounding box First translate every object into original position, then scale all by reciprocal value of scale factor (Note that scale factor and original position are stored into mesh object). Then it computes global bbox, scale every object, recalculate global bbox and finally translate every object in a right position.
